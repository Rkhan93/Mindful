Mindful

Your Solution to Project Managment

Is your mind full of all the tasks that need to be completed in work and at home? If so, Mindful is your solution.

Get started today

The application was developed to help you manager prorjects and tasks. As a users your projects, tasks, and their status can be entered, managed, and stored all on one place.

Product Features

This application allow you to enter each project you are working on or need to start working on. You are able to enter and track tasks associated with each project. The task tracking tool will break out your projects and tracking.

How to Use

To use this application, you will first need to create a log in. Once created, enter as many projects as needed. Within each project break out the tasks, owners, and due daes for each task within the project. As projects are work, update their status.
